## ðŸš€ Getting Started

### Prerequisites
- A modern web browser.
- A Google Gemini API Key (set in your environment as `API_KEY`).

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/luminacalc-ai.git
   ```
2. Navigate to the project folder:
   ```bash
   cd luminacalc-ai
   ```
3. Open `index.html` in your browser (preferably via a local server):
   ```bash
   npm install && npm start
   ```
